# React MF
A simple react micro frontend store

## Getting Started

### Prerequisites

- Node.js >= 16

### Installing

```
npm install
```

### Running

```
npm start
```

### Visualizing the app hierarchy

```
npx nx graph
```

## Built With

* [React](https://reactjs.org/) - The web framework used
* [Webpack](https://webpack.js.org/) - Module bundler
* [Lerna](https://lernajs.io/) - Monorepo management
